What is Css?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes HTML element are to be displayed on screen, paper,or in other media.
  3. Css is not neccesary CSS saves a lot of work. It can control the layout of multiple web pages all at once
  4. External stylesheet are stored in CSS files.

CSS can be added to HTML document in 3 Ways:

  • Inline-by using style attribute inside HTML elements
  • Internal-by using style element in the head section
  • External-by using a link element to link an external CSS files
  • The most common way to add CSS is to keep the style in external css files.However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.

When comparing CSS class vs id, the differnace is that css class applie a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use. a special URL to link directly to an elememt and it's used by javascipt.

In CSS, selectors are used to target a specific element or range of element on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID both are used to target ele,ments to which a style should be applied.
